# Tela de formulário
Este projeto é um formulário, onde valida as entradas do usuário e retorna se são válidas ou não e envia os dados em formato JSON para o console.

# O que foi utilizado:
 - React
 - TypeScript
 - Styled components
 - React-Hook-Form
 - ViteJS
 - React Icons
 - Yup
 - Yup-phone
 - React-Text-Mask
